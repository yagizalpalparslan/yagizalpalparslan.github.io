About Me

I’m a gameplay developer focused on building interactive 3D gameplay systems with Unity and C#. I work on player controls, game mechanics, object interactions, UI logic, and gameplay features that make projects feel responsive and playable.

Alongside Unity, I also have experience creating Unreal Engine projects using Blueprints, where I develop prototype mechanics, interaction systems, and level-based gameplay logic.

I’m more focused on the technical side of game development than game design. I enjoy turning gameplay ideas into working systems, solving technical problems, writing clean and functional code, and improving how the game feels through iteration.

I develop gameplay systems for Unity 3D projects, including player movement, camera behavior, object interactions, scoring systems, UI logic, and core game mechanics. I also create gameplay prototypes in Unreal Engine using Blueprints.

My focus is on coding, implementation, and making gameplay systems work smoothly inside the engine.
